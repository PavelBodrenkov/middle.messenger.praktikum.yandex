# Мессенджер
Проект находится в стадии разработки

## Функционал
 - Отправка сообщений
 - Отправка файлов, фото
 - Архивация чатов
 - Редактирование профиля

### Стэк
- PUG
- Parcel
- SCSS
- Express

### Установка
- `npm install` — установка зависимостей,
- `npm run dev` — запуск версии для разработчика,
- `npm run build` — сборка стабильной версии.
